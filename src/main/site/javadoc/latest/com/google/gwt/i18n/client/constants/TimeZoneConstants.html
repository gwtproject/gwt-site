com.google.gwt.i18n.client.constants
Interface TimeZoneConstants

All Superinterfaces:
Constants, Localizable, LocalizableResource
public interface TimeZoneConstants
extends Constants

TimeZoneConstants encapsulates a collection of time zone data for use with DateTimeFormat services. This class extends GWT's Constants class. The values of the constants are defined in property files named something like "TimeZoneConstants_xx.properties". Time zone data has only been provided for the "en" locale. Due to the large size of the time zone data for each locale, we recommend that applications retrieve the necessary data (i.e. over RPC) for the user's locale at run time.
