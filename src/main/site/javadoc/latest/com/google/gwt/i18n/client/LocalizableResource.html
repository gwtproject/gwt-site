public interface LocalizableResource
extends Localizable

This is the common superinterface to Messages and Constants. Each (and the Constants subinterface ConstantsWithLookup) provide compile-time localization of various forms of data. Messages is used for MessageFormat-style strings which can have parameters (including support for plural forms), while Constants can be other types, have simplified quoting requirements, and do not take any parameters. The annotations defined here are common to both -- see the individual subinterfaces for additional annotations which apply only to each one.
